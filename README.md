# 学生成绩管理系统

一个基于Python开发的学生成绩管理系统，提供完整的学生信息管理和成绩管理功能。

## 功能特性

### 1. 学生信息管理
- ✅ 添加学生（学号、姓名、年龄、性别、班级）
- ✅ 删除学生
- ✅ 修改学生信息
- ✅ 查询学生信息（支持按学号或姓名搜索）
- ✅ 显示所有学生列表

### 2. 成绩管理
- ✅ 录入成绩
- ✅ 修改成绩
- ✅ 删除成绩
- ✅ 查询学生成绩（显示总分和平均分）

### 3. 查询统计
- ✅ 学生成绩详情查看
- ✅ 班级统计（平均分、最高分、最低分）
- ✅ 单科统计
- ✅ 成绩排名（按平均分排序）

### 4. 数据持久化
- ✅ 使用JSON文件存储数据
- ✅ 自动保存和加载数据

## 系统要求

- Python 3.6 或更高版本
- 无需额外依赖包（仅使用Python标准库）

## 安装和使用

### 1. 下载文件

确保以下文件在同一目录下：
- `main.py` - 主程序文件
- `student_manager.py` - 学生管理模块
- `score_manager.py` - 成绩管理模块

### 2. 运行系统

在命令行中运行：

```bash
python main.py
```

### 3. 使用说明

系统启动后会显示主菜单，根据提示选择相应功能：

```
【主菜单】
1. 学生信息管理
2. 成绩管理
3. 查询统计
4. 退出系统
```

#### 学生信息管理
- 选择 `1` 进入学生信息管理子菜单
- 可以添加、删除、修改、查询学生信息

#### 成绩管理
- 选择 `2` 进入成绩管理子菜单
- 可以录入、修改、删除、查询成绩

#### 查询统计
- 选择 `3` 进入查询统计子菜单
- 可以查看学生详情、班级统计、成绩排名

## 数据存储

系统数据存储在 `students.json` 文件中，格式如下：

```json
{
  "学号": {
    "name": "姓名",
    "age": 年龄,
    "gender": "性别",
    "class": "班级",
    "scores": {
      "科目": 分数
    }
  }
}
```

## 使用示例

### 添加学生
1. 选择 `1` -> `1`（学生信息管理 -> 添加学生）
2. 依次输入学号、姓名、年龄、性别、班级

### 录入成绩
1. 选择 `2` -> `1`（成绩管理 -> 录入成绩）
2. 输入学号、科目、分数

### 查看排名
1. 选择 `3` -> `3`（查询统计 -> 成绩排名）
2. 系统会显示所有学生的平均分排名

## 注意事项

1. 学号必须唯一，不能重复
2. 分数范围：0-100分
3. 删除学生时会同时删除该学生的所有成绩
4. 数据会自动保存，无需手动保存

## 文件说明

- `main.py` - 主程序，包含用户界面和菜单系统
- `student_manager.py` - 学生信息管理模块，负责学生的增删改查
- `score_manager.py` - 成绩管理模块，负责成绩的录入、统计等功能
- `students.json` - 数据存储文件（运行后自动生成）
